前言

隨著專案規模成長，資料庫結構的管理與版本控制變得愈發重要。Alembic 是一個專為 SQLAlchemy 設計的資料庫遷移工具，可協助我們有系統地管理 PostgreSQL 資料庫結構的變更。本篇文章旨在整理 Alembic 新手入門流程，並提供步驟說明與範例，協助讀者快速上手與解決常見疑問。

處理問題

主要目標為：

  1. 釐清 Alembic 的安裝與初始化步驟。
  2. 介紹如何設定 PostgreSQL 連線。
  3. 示範建立、編輯與套用 Migration 的流程。
  4. 補充常用指令與注意事項，降低新手學習障礙。

6. 常用 Alembic 指令整理

指令 說明
alembic revision -m "描述" 產生新的 migration 檔案
alembic upgrade head 套用所有尚未執行的 migration 到最新狀態
alembic downgrade -1 回復上一個 migration（可多次執行回朔）
alembic current 顯示目前資料庫的 migration 狀態
alembic history 顯示所有 migration 歷史紀錄
alembic show <revision> 顯示指定 migration 的詳細內容
