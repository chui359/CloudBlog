前言

本文說明如何用 Python 的 sorted() 和 lambda 實現列表的置頂與排序功能，並搭配範例與運作邏輯說明。

問題

在物件列表排序時，常遇到這兩個需求：

  • 讓指定條件（如 pinned=True）的物件「置頂」
  • 在置頂及未置頂群組內，依其他屬性（如名稱或建立日期）排序

如何同時達成這兩個目標？以下提供解決方案。

解決辦法

利用 sorted() 的 key 參數，回傳一個元組：

  • 第一元素用於置頂
  • 第二元素用於群組內排序

程式碼邏輯說明

  • Project 類別：名稱、建立日期、是否置頂
  • projects：物件列表
  • sortKey、sortScend：決定排序屬性與方向
  • sort_key_map：前端參數與物件屬性對應
  • sorted()：
    • key=lambda p: (not p.pinned, getattr(p, sort_attribute))
      第一個元素讓置頂在前，第二個元素則依指定屬性排序
    • reverse=reverse_order
      控制排序方向（升冪或降冪）

運作邏輯

  1. 先以 not p.pinned 分群，pinned=True 會排在最前面
  2. 群組內再依選定屬性排序（如名稱、日期）
  3. 排序方向可控制（升冪或降冪）
